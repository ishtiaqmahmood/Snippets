# C++ and JavaScript Programming Examples

A comprehensive collection of programming examples in C++ and JavaScript, covering everything from basic syntax to object-oriented programming (OOP) and data structures. This repository serves as a learning resource for developers looking to master the fundamentals of both languages.

## 📁 Repository Structure

```text
.
├── c++/                # C++ source files covering basics to advanced topics
│   └── oop/           # Deep dive into Object-Oriented Programming in C++
└── javascript/        # JavaScript fundamentals and core concepts
```

## 🚀 Getting Started

### Prerequisites

- **C++**: A C++ compiler like `g++` (GCC) is required.
- **JavaScript**: [Node.js](https://nodejs.org/) is recommended for running the scripts in a terminal environment.

### Running C++ Examples

1. Navigate to the `c++/` directory.
2. Compile a file using `g++`:
   ```bash
   g++ hello.cpp -o hello
   ```
3. Run the executable:
   ```bash
   ./hello
   ```

### Running JavaScript Examples

1. Navigate to the `javascript/` directory.
2. Run a file using `node`:
   ```bash
   node variables.js
   ```

## 📚 Topics Covered

### C++ Examples

#### 🔹 Basic Syntax & Control Flow
- **Basics**: Hello World, User Input, Formatting Output.
- **Operators**: Arithmetic, Assignment, Bitwise, Unary, Conditional, Special Operators.
- **Control Flow**: `if-else` (Selection), `switch` (Digit spelling, Vowel/Consonant), `for` loop, `while` loop, `do-while` loop.
- **Problem Solving**: Leap Year, Even/Odd, Large/Small number, Letter Grade.

#### 🔹 Object-Oriented Programming (OOP)
- **Fundamentals**: Classes, Objects, `this` keyword, Scope Resolution.
- **Constructors**: Basic Constructors, Constructor Initializers.
- **The Four Pillars**:
  - **Encapsulation**
  - **Abstraction**
  - **Inheritance**
  - **Polymorphism** (Function Overriding, Virtual Functions).
- **Advanced OOP**: Friend Classes, Function Overloading.

#### 🔹 Algorithms & Mathematics
- **Math**: Prime Numbers, Fibonacci Series, Factorial, Armstrong Number, Strong Number, Palindrome.
- **Calculators**: GCD/LCM, Temperature Converter, Area of Triangle, Multiplication Table.
- **Series**: Sum of n numbers, Sum of Even/Odd numbers, Square series, and various multiplication series.
- **Arrays**: 2D Arrays, User Input in Arrays, Passing Arrays to Functions.

#### 🔹 Advanced Concepts
- **Memory**: Pointers, Adding numbers using pointers.
- **Functions**: Function Templates, Recursion, Return Types, Default Parameters.
- **I/O**: File Handling (Read/Write), String Class, String Demo.
- **Error Handling**: Exception Handling.

### JavaScript Examples

#### 🔹 Fundamentals
- **Variables**: `var`, `let`, `const`, Multi-word variables.
- **Data Types**: Strings, Numbers, Booleans, etc., and Type Conversion.
- **Output**: Console logging techniques.

#### 🔹 Control Structures
- **Conditional**: `if-else` statements, `switch` cases.
- **Iteration**: Various types of loops.

#### 🔹 Data Structures & Objects
- **Arrays**: Methods and manipulations.
- **Objects**: Object Literals and property access.
- **Built-in Objects**: Math object, Date and Time manipulation.

#### 🔹 Functions & Scope
- **Functions**: Declarations and usage.
- **Scope**: Block scope vs. Global scope.
- **Environment**: Working with the `window` object.

## 🛠 Tools & Technologies
- **Languages**: C++ (C++11 or later), JavaScript (ES6+).
- **Compilers/Runtimes**: GCC/G++, Node.js.
